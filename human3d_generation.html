Large Language Models in 3D Human Generation

LLMs Powered Personalized Text and Image to 3D Human Body Model Pipeline

We propose a pipeline with an interactive interface for generating and modifying 3D human body models using text or image input. The pipeline includes text-to-3D model generation, image-to-3D model generation, and text-controlled modifications of 3D models utilizing parametric models SMPL and SMPL-X. We extract body parameters from models, label their meanings and relationships, and use a specialized version of ChatGPT to translate natural language descriptions into body parameters. These body parameters are input into the generation model to create 3D human models. We use the SMPL-X API from Meshcapade Me for image-to-3D model generation.

Modifications to the 3D models are achieved by extracting and standardizing body measurements, then using Multivariable Linear Regression to map them to body parameters for model generation. We avoided direct generation from body measurements to prevent gaps in the models. The mapped parameters are fed into a specialized ChatGPT, enabling users to modify the 3D model based on natural language descriptions. We also created a web-based interface to give users easy access to the entire pipeline and detailed explanations of each step in the model generation process. Finally, we evaluate the quality of the generated models using a survey. Our contributions are summarized as follows:

  • Proposed a complete pipeline with an interactive interface that allows users to generate 3D human body models based on text or image input and modify the models based on text input.
  • Proposed a mapping method to transform body measurements into body parameters for improved 3D model generation.
  • Developed a specialized ChatGPT for translating natural language descriptions and instructions into body parameters in the SMPL model.
  • Demonstrated the potential and limitations of using LLMs for 3D human body model generation to encourage exercise and outlined future research directions.

Text and Image to Model Generation

Our system's 3D human body models are categorized into three sets. The first set (a) includes models created from simple descriptive terms, such as "thin," "strong," or "overweight." The second set (b) uses more detailed descriptions, incorporating specific attributes like height, weight, and body characteristics. The third set (c) are models generated directly from images.

Text Controlled 3D Human Body Model Modification

This figure demonstrates how the system modifies 3D human body models based on different types of text inputs. The system takes original images and processes inputs such as preserving body shape, making direct text-based changes (e.g., "thinner" or "stronger"), and predicting results from short-term (one month) and long-term (one year) exercise plans.

Evaluation

Due to the lack of comprehensive human body data, the wide variety of body types, and people's different interpretations of body-related features, we could not establish a clear baseline for comparison. Instead, we conducted a subjective evaluation using surveys to gather user feedback on whether the generated models met their expectations.

This figure presents the results for four survey questions, where participants were asked to select the description that best matches the shown model. (a) and (b) illustrate the results of simple adjectives, while (c) and (d) demonstrate more complex descriptions. The highlighted descriptions (orange) in the chart correspond to those used to generate the 3D body model. The results indicate that participants' opinions are more consistent for simpler descriptions, while responses become more diverse for more complex descriptions.

This figure presents the results for two questions. The model at the bottom is transformed from the one at the top. Participants were asked to identify the activity most likely responsible for the change. The highlighted descriptions (orange) in the chart correspond to the activity used to transform the 3D body model. The results indicate that participants can recognize changes resulting from exercise but are less sensitive to the specific effects of different types of exercise, as shown in (a).

Conclusion

In conclusion, we present a comprehensive pipeline for generating and modifying 3D human body models using text and image inputs, combining parametric models with LLMs. We redefined and labeled the body parameters in the parametric models and trained a specialized version of ChatGPT to translate natural language descriptions into body parameter sets or adjustments. For text-based model modification, we designed a mapping method that transforms body measurements into body parameters to support the modification process better. By introducing an interactive interface, we offer an accessible and customizable approach to 3D body modeling. This development highlights the potential of LLMs to support more personalized and motivational applications in areas like fitness, where users can visualize potential physical improvements. In this context, our survey evaluates the feasibility and limitations of using LLMs, which lays the foundation for future research on enhancing user engagement and model precision. In the future, we aim to improve the pipeline by implementing everything locally and gathering more human body data to enhance the model's accuracy.
